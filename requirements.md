## Requirements

To successfully run this platform, you need to install the following packages:

- pandas
- numpy
- matplotlib
- csv
- json
- reportlab
- os
- tempfile
- datetime
- decimal
- string
- flask
- flask_socketio
- time
- werkzeug.utils
- win32com.client
- pythoncom
- random
- shutil
- bleach
- numpy

**Note:** In order to use the COM Library successfully, you need to install Office 365 and have a valid license.
